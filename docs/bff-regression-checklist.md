# BFF 전환 회귀 테스트 체크리스트

전환 단계별로 아래 항목을 최소 한 번 이상 점검합니다.

## P0 (인증/세션)
- [ ] 로그인 성공/실패 처리
- [ ] 로그아웃 후 세션 정리(쿠키 제거/캐시 초기화)
- [ ] 토큰 만료 후 재발급 성공/실패 처리

## P1 (쓰기/변경)
- [ ] 게시글 생성/수정/삭제
- [ ] 댓글 생성/삭제
- [ ] 좋아요 추가/취소
- [ ] 이미지 업로드 성공/실패 처리

## P2 (개인화 읽기)
- [ ] 내 프로필 조회
- [ ] 내 활동 통계 조회
- [ ] 내 게시글/댓글/좋아요 목록 조회
- [ ] 비로그인/권한 없는 사용자 처리

## P4 (공개 읽기)
- [ ] 게시글 목록 로딩
- [ ] 게시글 상세 로딩
- [ ] 태그/인기글/검색 로딩
